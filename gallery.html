Model #1

Earpods Headphones

This model is all about showcasing the practical use of the Sweep Feature in CAD design. My journey with this project began with a curiosity to understand 3D sketches and how they could bring life to a Sweep Feature.

My starting point was crafting a Headphone Jack, a humble beginning that required a simple sketch and the Revolve Feature. "It felt like taking the first steps on a learning journey.

To create the essential connection, I employed the Extrude Feature, shaping a short, straight cable piece. It was like laying the foundation of a bridge between components.

As the project progress, I delved into creating a stress relief section, using the Extrude Feature with a twist - I set the Draft parameter and left Merge unchecked. This approach allowed for precise cuts from top to bottom, keeping the cable separate as individual pieces.

With all the parts ready, I moved on to the most exciting part of the journey. I utilized the 3D sketch I had crafted earlier and the cable profile to seamlessly connect all elements using the Sweep Feature. It was like watching puzzle pieces fall into place, and I learned the importance of tangency relations to make the connections smooth and elegant.

At the end, I managed to utilize acquired knowledge and create a simple electrical socket plug with a cable using the same technique

After completing this project, I felt a sense of accomplishment and decided to continue my learning journey through practice.


A Soda Bottle

This model demonstrates how to use the Revolve and Cavity Features in CAD design. My journey started by creating a revolve sketch profile, which became the main body of the bottle.

After crafting the bottle's main body, I decided to add some details. I used Sweep Cut and Circular patterns to create attractive indents inside the bottle.

For the neck of the bottle, I saw it as a great opportunity to learn and use the Helix feature. This helped me create a thread-like design.

Once the body was successfully made, I used the Shell feature and turned it into an assembly. Using the original part as a guide, I created a new part for the bottle's cap.

To give the bottle a realistic touch, I added a transparent appearance. With the help of the Cavity Feature, I was able to simulate the liquid inside the bottle and make it look accurate.

As a final touch, I learned how to use decals and applied them to both the cup and the bottle body to create labels.

This has been a very interesting journey. I've learned that even complex shapes, at first glance, are achievable. I can't wait to showcase more in the future.


Computer Mouse

In the model above, I practiced using the Loft and Split Features.

I began by creating multiple profiles for the Loft feature. Once the Loft was completed, I added some fillets to finalize the basic solid shape.

With the basic shape in good order, I decided to Shell the model and then split it into two parts, allowing me to work on each side separately.

This workflow enabled me to add more details to the bottom part. I focused on incorporating mounting bosses for screws and space for a PCB, then proceeded to cut holes for the screws.

Returning to the top part was straightforward; it simply involved switching windows. This experience highlighted how efficient SolidWorks can be, thanks to its built-in features that streamline the modeling process.

Working on the top part presented some challenges. I had to make cuts for operational buttons and add a PCB with its major components, including the Scroll Wheel and button casings.

Once the PCB was successfully incorporated, I knew the exact clearances needed to make the final cut in the top section. To wrap it up, I added screws and applied appearances to give my model a polished look. This project served as a valuable lesson in utilizing the Loft feature, which will enhance my future modeling endeavors.

A Game Controller - Simplified

Above is a simplified version of a Game Controller.

The model was first created using simple Extrude Features to establish the rough shape.

Then, with the Filler Feature, I refined the shape into a more detailed solid.

Subsequently, the focus shifted to creating the shell and adding details. A challenging aspect was ensuring that the angles were not too tight to make the Shell Feature work correctly.

For this, I found the Evaluate Toolkit to be immensely helpful. With appropriately adjusted curvatures, I was able to successfully create the shell, adding final touches such as analogs and button cuts.

Utilizing Extrude Features with the Thin parameter allowed me to create inner ribs, enhancing the model's structure.

Another opportunity to apply the Evaluate Toolkit arose when I realized that my model wouldn't be optimal for manufacturing if I used a regular straight line for splitting. So, with the assistance of the Parting Line Analysis Tool, I managed to create a suitable splitting line.

Following these steps, the remaining tasks were relatively straightforward - adding the PCB with buttons and analogs.

In reality, a fully detailed model would require much more complexity, but as it was a simplified version, I could utilize various tools to create it relatively quickly while thoroughly enjoying the process.

Conducting Draft Analysis and applying Draft Features at the end ensured that the part could be manufactured using Injection Molding Technology, which I found to be an intriguing aspect to explore further.

I found my SolidWorks journey very interesting and appealing, motivating me to study more and explore the endless possibilities. I can't wait to improve my skills by creating more models, hopefully for your company.
